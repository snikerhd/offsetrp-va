# Vehicle Impound Report Generator - OFFSET Portugal RP

A police terminal application for generating vehicle impound reports with license plate simulation, penalty code tables, and session history management.

## Features

- **Department Selection**: Switch between DPSA (San Andreas), DPLS (Los Santos), and DBC (Blaine County)
- **Real-time License Plate Simulator**: Visual Portuguese/EU style license plate preview
- **Penalty Code Table**: Pre-defined infractions with associated fines
- **Advanced Mode**: Additional fields for officer info, location, and custom fines
- **Session History**: Local storage persistence with search functionality
- **Audio Feedback**: Interactive sounds for button clicks and notifications
- **Export Options**: Download individual reports or compile entire session

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)
- Web Audio API (custom sounds)
